网格布局

Grid 布局是 CSS 中最强大的二维布局系统（同时控制行和列），相比 Flex 布局（一维），它能更精准地控制页面的整体结构，特别适合复杂的网格型布局（比如电商首页、仪表盘、卡片墙等）

一、核心概念

术语通俗解释
容器（Grid Container）开启 display: grid 的父元素，所有子元素会成为网格项
项目（Grid Item）容器的直接子元素（孙子元素不算）
网格线（Grid Line）分割网格的线（横向 = 行线，纵向 = 列线），比如 3 列会有 4 条列线
网格轨道（Grid Track）两条网格线之间的区域（行轨道 = 行高，列轨道 = 列宽）
网格单元格（Grid Cell）行和列交叉形成的最小单位（类似表格的单元格）
网格区域（Grid Area）多个单元格合并后的区域

二、基础使用步骤

  1. 开启 Grid 布局

先给父容器设置 display: grid，这是一切的基础：

/* 容器 */
.grid-container {
  display: grid; /* 块级 Grid，占满一行 */
  /* display: inline-grid; 行内 Grid，宽度由内容决定 */
}

/* 项目（容器的直接子元素） */
.grid-item {
  /* 项目的样式，比如背景、内边距 */
  background: #f0f0f0;
  padding: 20px;
  text-align: center;
}
  1. 定义网格的行列（核心）

通过 grid-template-columns（列）和 grid-template-rows（行）定义网格的轨道大小，这是控制网格结构的关键。

（1）固定尺寸 + 弹性尺寸

.grid-container {
  display: grid;
  /* 定义 3 列：第一列 100px，第二列 200px，第三列 150px */
  grid-template-columns: 100px 200px 150px;
  /* 定义 2 行：第一行 80px，第二行 100px */
  grid-template-rows: 80px 100px;
  /* 行列间距（可选） */
  gap: 10px; /* 等同于 row-gap: 10px; column-gap: 10px; */
}

.grid-item {
  /* 项目的样式，比如背景、内边距 */
  background: #f0f0f0;
  padding: 20px;
  text-align: center;
}

（2）弹性布局（最常用）
使用 fr 单位（分数单位），自动分配剩余空间：

.grid-container {
 display: grid;
  /* 3 列，每列占 1/3 宽度 */
  grid-template-columns: 1fr 1fr 1fr;
  /* 简写：repeat(3, 1fr) */
  grid-template-columns: repeat(3, 1fr);
  
  /* 混合使用：第一列 200px，剩余两列平分 */
  grid-template-columns: 200px 1fr 1fr;
  
  /* 行：第一行自动高度，第二行占 2 份，第三行占 1 份 */
  grid-template-rows: auto 2fr 1fr;
  
  gap: 15px;
}

.grid-item {
  /* 项目的样式，比如背景、内边距 */
  background: #f0f0f0;
  padding: 20px;
  text-align: center;
}

（3）响应式网格（自动换行）
使用 auto-fit/auto-fill + minmax() 实现自适应网格（无需媒体查询）：

.grid-container {
  width: 600px;
  display: grid;
  /* 每列最小 200px，最大 1fr，自动换行 */
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 10px;
}

.grid-item {
  /* 项目的样式，比如背景、内边距 */
  background: #f0f0f0;
  padding: 20px;
  text-align: center;
}
  • auto-fit：尽可能多的列，空白空间分配给现有列；
  • auto-fill：尽可能多的列，即使列是空的（留白）。
