网格布局

Grid 布局是 CSS 中最强大的二维布局系统（同时控制行和列），相比 Flex 布局（一维），它能更精准地控制页面的整体结构，特别适合复杂的网格型布局（比如电商首页、仪表盘、卡片墙等）

一、核心概念

术语 通俗解释
容器（Grid Container） 开启 display: grid 的父元素，所有子元素会成为网格项
项目（Grid Item） 容器的直接子元素（孙子元素不算）
网格线（Grid Line） 分割网格的线（横向 = 行线，纵向 = 列线），比如 3 列会有 4 条列线
网格轨道（Grid Track） 两条网格线之间的区域（行轨道 = 行高，列轨道 = 列宽）
网格单元格（Grid Cell） 行和列交叉形成的最小单位（类似表格的单元格）
网格区域（Grid Area） 多个单元格合并后的区域
